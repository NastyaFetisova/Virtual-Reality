# 🌐 Virtual Reality Landing Page

> Современный одностраничный сайт, посвящённый технологиям виртуальной реальности. Проект демонстрирует навыки вёрстки современных веб-интерфейсов с использованием CSS Flexbox и визуальных эффектов (градиенты, блюр, клип-паты, неоновая эстетика).

## ✨ Особенности

- ✅ Полностью адаптивный дизайн
- ✅ Современная тёмная тема с неоновыми акцентами
- ✅ Сложные формы и геометрические маски (`clip-path`)
- ✅ Эффекты размытия и стекла (`backdrop-filter`)
- ✅ Градиентные рамки и кнопки
- ✅ Кастомные стили для карусели отзывов
- ✅ Собственные CSS-переменные и модульная структура
- ✅ Псевдоэлементы и декоративные слои
